🎨 Appearance
Light mode is easier to read outdoors in daylight. AMOLED uses a true-black background — on OLED screens black pixels switch fully off, saving battery at night. Auto follows your phone/PC's system theme. Your choice is remembered on this device.
👤 Profiles
Each profile keeps its own completely separate data — expenses, plans, history, even its own Cloud Sync code. Great for a partner's budget or a business wallet. Switch anytime from the dropdown in the header.
📱 Payment QR codes
Save your QR Ph code from each wallet app (Maya, GoTyme, MariBank, RCBC…) as an image, then add it here. Tap 📱 My QR on the CC Split or Hangout page to show it full-screen for friends to scan. Any QR Ph code works from any InstaPay app.
☁️ Cloud Sync
Pick a sync code — any secret phrase you'll remember. Enter the same code on your other devices and they'll all share the same data. Make it long and unique (like a password) — anyone who knows it can see your data.
💾 Storage used
Everything lives in this browser's local storage (usually capped around ~5 MB). Wallet QR images and long histories take the most space — export a backup before it fills up.

📤 Export data
Downloads a .json file with everything — savings, bills, goals, tracker, split history. Copy it to another device or save as backup.
📥 Import data
Restore from a backup file. This replaces all current data on this device and reloads the app.
